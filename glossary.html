Задания

Языки программирования: Средства выражения алгоритмов в виде программ, написанных на конкретных языках, таких как C++, Java, Python и другие.
Алгоритм: Последовательность шагов для решения задачи.
Натуральные числа: Положительные целые числа, начиная с 1.
Ряд Фибоначчи: Числовой ряд, где каждое следующее число равно сумме двух предыдущих (например, 1, 2, 3, 5, 8, ...).
Простые числа: Числа, имеющие только два делителя: 1 и само число.
